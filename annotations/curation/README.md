# Curation Working Files

This directory holds working lists used to choose what becomes reviewed teaching content.

These files are not final viewer annotations. They are queues and crosswalks that help us decide:

- which atlas labels are useful as-is,
- which labels need learner-facing renaming,
- which high-yield structures need manual annotation, and
- which structures need an external atlas or additional sequence before they are reliable.

The reviewed teaching database will live under `annotations/structures/`.
