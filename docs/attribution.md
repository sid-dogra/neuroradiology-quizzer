# Attribution And Licensing Notes

## FreeSurfer

FreeSurfer-derived labels and outputs should be attributed clearly if used as a basis for annotations.

Useful links:

- FreeSurfer license: https://surfer.nmr.mgh.harvard.edu/fswiki/FreeSurferSoftwareLicense
- FreeSurfer recon-all: https://surfer.nmr.mgh.harvard.edu/fswiki/recon-all
- FreeSurfer color lookup table: https://surfer.nmr.mgh.harvard.edu/fswiki/FsTutorial/AnatomicalROI/FreeSurferColorLUT

## NiiVue

NiiVue is the current web viewer foundation for static NIfTI images and overlays.

Useful links:

- NiiVue: https://niivue.com/
- NiiVue GitHub: https://github.com/niivue/niivue

## OHIF

OHIF remains a possible future viewer option for DICOM/DICOMweb or DICOM SEG workflows.

Useful links:

- OHIF docs: https://docs.ohif.org/
- OHIF deployment: https://docs.ohif.org/deployment/
